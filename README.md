# HacktobeRfest

**EN**
Another edition of the [hacktober fest](https://hacktoberfest.digitalocean.com/) has started but, as expected, among the project suggestions for contribution, we did not find R language repositories. To advertise these projects, we created this repository to list such projects, and thus make them accessible to the programmers community in the event.
